Introducing Qwikifox!

BETA

Qwkifox (beta) is a clean search engine proudly made in India! This works with official Wikipedia, so nothing to worry about!. It has a good and clean user interface with Dark, Light and custom themes!

Getting Started

To get started, click the Launch button and Start searching! The Search features have awesome search suggestions and you can also search for a specific article. There is also Option to change theme. You can also Import custom css themes.

Features

This also has some awesome small features such as : Searching articles, Bookmarking Articles, Downloading articles as PDF, Instant Sharing, Sharing Quotes, Social Sharing, Fully customizable UI + Themes and much more than you can imagine!
